Рџа№ИЈ рЃЊрЃўрЃАрЃЎрЃџрЃћрЃўрЃЏрЃћрЃарЃў / рЃњрЃљрЃюрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ
рЃћрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃБрЃЏрЃЮрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃЊрЃљ рЃАрЃљрЃбрЃўрЃарЃБрЃџрЃў рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃњрЃў рЃљрЃа рЃўрЃАрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃўрЃќрЃюрЃљрЃЊ рЃарЃљрЃўрЃЏрЃћ рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃўрЃА, рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃљрЃю рЃбрЃарЃљрЃњрЃўрЃЎрЃБрЃџрЃў рЃцрЃљрЃЦрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃљрЃЏрЃљрЃарЃЌрЃџрЃћрЃЉрЃљ, рЃўрЃЊрЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃљрЃю рЃАрЃћрЃарЃўрЃЮрЃќрЃБрЃџрЃљрЃЊ рЃњрЃљрЃюрЃ«рЃўрЃџрЃЋрЃљ.

рЃАрЃбрЃљрЃџрЃўрЃюрЃўрЃА рЃЊрЃљ РђюрЃЊрЃљрЃ«рЃЋрЃарЃћрЃбрЃўрЃАРђЮ рЃЌрЃћрЃЏрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃБрЃарЃљрЃфрЃ«рЃДрЃЮрЃцрЃўрЃА рЃљрЃю рЃФрЃљрЃџрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃгрЃљрЃ«рЃљрЃџрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ Рђћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃ«рЃБрЃЏрЃарЃЮрЃЉрЃўрЃА, рЃљрЃЉрЃАрЃБрЃарЃЊрЃўрЃА рЃЊрЃљ рЃЏрЃћрЃЏрЃћ рЃЎрЃБрЃџрЃбрЃБрЃарЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў.

рЃЌрЃБ рЃћрЃА рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃЮ рЃљрЃю рЃерЃћрЃБрЃарЃљрЃфрЃ«рЃЏрЃДрЃЮрЃцрЃћрЃџрЃўрЃљ, рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃЮрЃЌ, рЃарЃЮрЃЏ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃарЃљрЃўрЃЏрЃћрЃА рЃАрЃћрЃарЃўрЃЮрЃќрЃБрЃџрЃљрЃЊ рЃљрЃдрЃЦрЃЏрЃљ Рђћ рЃћрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃерЃћрЃ«рЃћрЃЊрЃБрЃџрЃћрЃЉрЃљрЃљ рЃўрЃарЃЮрЃюрЃўрЃљрЃќрЃћ рЃЊрЃљ рЃърЃљрЃарЃЮрЃЊрЃўрЃљрЃќрЃћ.

рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃЏрЃўрЃўрЃдрЃЮрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃўрЃБрЃЏрЃЮрЃарЃўрЃЌ, рЃЊрЃљ рЃљрЃарЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃарЃћрЃљрЃџрЃБрЃарЃў рЃерЃћрЃ«рЃћрЃЊрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃљрЃю рЃЏрЃЮрЃарЃљрЃџрЃБрЃарЃў рЃњрЃќрЃљрЃЋрЃюрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃЏрЃАрЃљрЃ«рЃЋрЃћрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў.



­Ъфќ рЃАрЃбрЃљрЃџрЃўрЃюрЃўрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃбрЃЮрЃарЃў РђЊ рЃўрЃБрЃЏрЃЮрЃарЃўрЃАрЃбрЃБрЃџрЃў рЃџрЃўрЃАрЃб-рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃў
рЃАрЃбрЃљрЃџрЃўрЃюрЃўрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃбрЃЮрЃарЃў рЃљрЃарЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў Python рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў PyQt5-рЃќрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮ, рЃгрЃљрЃерЃљрЃџрЃЮ рЃљрЃю рЃАрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃЮ РђърЃЊрЃљрЃАрЃљрЃ«рЃЋрЃарЃћрЃбрЃљрЃЊ рЃњрЃљрЃюрЃгрЃўрЃарЃБрЃџрЃў рЃАрЃБрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАРђю рЃАрЃўрЃљ. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃўрЃБрЃЏрЃЮрЃарЃўрЃАрЃбрЃБрЃџ рЃърЃљрЃарЃЮрЃЊрЃўрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃАрЃћрЃарЃўрЃЮрЃќрЃБрЃџрЃЮрЃЉрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃљрЃЦрЃфрЃћрЃюрЃбрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃўрЃарЃЮрЃюрЃўрЃљрЃАрЃљ рЃЊрЃљ "meme" рЃЎрЃБрЃџрЃбрЃБрЃарЃљрЃќрЃћ.

­ЪД░ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:
РъЋ рЃЏрЃўрЃБрЃАрЃљрЃ»рЃћ рЃЊрЃљрЃ«рЃЋрЃарЃћрЃбрЃљ РђЊ рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃЌ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ РђърЃАрЃБрЃЉрЃўрЃћрЃЦрЃбрЃўРђю рЃАрЃўрЃљрЃерЃў

РЮї рЃЊрЃљрЃ«рЃЋрЃарЃўрЃбрЃћ рЃАрЃБрЃЉрЃўрЃћрЃЦрЃбрЃў РђЊ рЃАрЃўрЃўрЃЊрЃљрЃю рЃгрЃљрЃерЃљрЃџрЃћ рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃърЃўрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљ

­ЪћЦ рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃљрЃ«рЃЋрЃарЃўрЃбрЃћ РђЊ рЃњрЃљрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћ рЃАрЃўрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ

Рўа№ИЈ рЃЏрЃЮрЃгрЃљрЃЏрЃџрЃћ рЃАрЃбрЃљрЃџрЃўрЃюрЃў РђЊ рЃЊрЃљрЃ«рЃБрЃарЃћ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ ­Ъўё



­ЪЏа рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў:
рЃћрЃюрЃљ: Python 

рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ: PyQt5
